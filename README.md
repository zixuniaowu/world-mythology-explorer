# 🌍 World Mythology Explorer / 世界神話エクスプローラー

An interactive web application for exploring world mythologies through immersive readers, games, and animations.

## 🎯 Overview

World Mythology Explorer is a comprehensive platform that brings ancient mythologies to life through modern web technologies. The project features interactive mythology readers, educational games, and animated storytelling for various cultural mythologies including Chinese, Japanese, Greek, Norse, Egyptian, Indian, and Mesopotamian traditions.

## ✨ Features

### 🗺️ Interactive World Map
- Visual exploration of global mythologies
- Click on regions to discover their mythological heritage
- Beautiful animated interface with particle effects

### 📚 Mythology Systems

#### 🇨🇳 Chinese Mythology
- **Shanhaijing (山海經)**: Complete reader with traditional and modern interpretations
- **Fengshen Yanyi (封神演義)**: Interactive story reader
- **Liaozhai Zhiyi (聊齋誌異)**: Strange tales collection
- Games and animations for each mythology

#### 🇯🇵 Japanese Mythology  
- **Kojiki (古事記)**: Japan's oldest chronicle with complete text
- **Nihon Shoki (日本書紀)**: Chronicles of Japan
- **Yokai (妖怪)**: Japanese folklore creatures
- Interactive games and animations

#### 🏛️ Other Mythologies
- Greek Mythology
- Norse Mythology
- Egyptian Mythology
- Indian Mythology
- Mesopotamian Mythology

### 🎮 Interactive Features
- **Mythology Games**: Educational games for each mythology system
- **Animated Stories**: Visual storytelling with traditional art styles
- **AI-Powered Tools**: Text-to-animation converter for creating custom mythology animations

## 🚀 Getting Started

### Prerequisites
- Python 3.x (for running local server)
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/world-mythology-explorer.git
cd world-mythology-explorer
```

2. Start a local server:
```bash
python3 -m http.server 8080
```

3. Open your browser and navigate to:
```
http://localhost:8080
```
