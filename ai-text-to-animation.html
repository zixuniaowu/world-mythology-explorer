Local Model Setup (Ollama)

1. Install Ollama: curl -fsSL https://ollama.ai/install.sh | sh

2. Pull a model: ollama pull llama2

3. Run with CORS: OLLAMA_ORIGINS=* ollama serve

4. Select "Ollama (Local)" and enter endpoint
