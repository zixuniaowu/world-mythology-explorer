ブラウザキャッシュのクリア方法

注意： index.htmlは正常に更新されていますが、ブラウザがキャッシュされた古いバージョンを表示しています。

方法 1: ハードリフレッシュ（推奨）

以下のキーボードショートカットを使用してください：

  • Windows/Linux: Ctrl + Shift + R または Ctrl + F5
  • Mac: Cmd + Shift + R

方法 2: 開発者ツールを使用

  1. F12キーを押して開発者ツールを開く
  2. ネットワークタブに移動
  3. 「キャッシュを無効化」にチェックを入れる
  4. ページを再読み込み（F5）

方法 3: 直接新しいページにアクセス

以下のリンクをクリックして、タイムスタンプ付きでアクセス：

新しいindex.htmlを開く

方法 4: ブラウザのキャッシュを完全にクリア

Chrome/Edge:

  1. 設定 → プライバシーとセキュリティ
  2. 閲覧履歴データの削除
  3. 「キャッシュされた画像とファイル」にチェック
  4. 削除をクリック

Firefox:

  1. 設定 → プライバシーとセキュリティ
  2. Cookieとサイトデータ → データを消去
  3. 「ウェブコンテンツのキャッシュ」にチェック
  4. 消去をクリック

以下のページで新しいデザインを確認できます：

  • index.html（メインページ）
